# srg.id.au (astro)

an astro recreation of shaunak gadkari's personal site, migrated locally from the original zola codebase.

## local development

```bash
npm install
npm run dev
```

## production build

```bash
npm run build
npm run preview
```

## project notes

- source content and assets were migrated from the original `portfolio-zola` repo
- this repo is intended to be a standalone astro implementation of the site
- public assets live in `public/`
- blog content lives in `src/content/posts/`
